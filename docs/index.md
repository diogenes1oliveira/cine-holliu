# cine-holliu

> A drugless trip through the Fundamentals of Prompt Engineering

## Quick Start

1. Install dependencies: `uv sync`
2. Serve documentation: `just dev`
3. Build documentation: `just build`
4. Convert markdowns: `just md2html`
